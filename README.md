Для использования на Kaggle можно использовать приведенный ноутбук. Необходимо:
1. Выбрать 2xT4 GPU (vllm serve логика оптимизирована под них)
2. Указать собственные API ключи для Яндекс Карт и ORS (оба доступны бесплатно)
3. Залить картинки животных в качестве датасета, указывать к ним путь
Из-за отсутствия личной Nvidia GPU гарантируется корректное выполнение кода только внутри Kaggle ноутбука
